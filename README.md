
Example Web-Api Typescript and Swagger

# Install project in local workstation

## 1. clone project
```bash
git clone https://github.com/jundat95/tn.sevice.test.git
```

## 2. go to folder
```bash
cd tn.sevice.test
```

## 3. install package
```bash
npm i
```

## 4. run app
```bash
npm run build
npm run start
```

## 5. go to swagger with link: http://localhost:3000/docs/
